LUCAR

Software de identificação de viaturas

Sistema inteligente integrado de registo e identificação automática de veículos por reconhecimento óptico de matrículas

Link do projeto no Github
Link do projeto
O presente sistema é sobre dois módulos de software integrados, que juntos possibilitam o acesso a dados proprietários de viaturas.
Um módulo é WEB e permite registar em um banco de dados os dados da viatura e do proprietário da viatura,de modo que seja possível ver o histórico geral de proprietários que uma viatura já teve desde que foi concebida.
Outro módulo é mobile e permite a identificação automática de veículos por reconhecimento óptico de matrículas, e também retorna os dados da viatura, do proprietário atual e o histórico de todos os proprietário que a viatura já teve.
